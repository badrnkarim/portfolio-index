# Security Policy

## Supported Versions
This repository is maintained as a portfolio project. The **main branch** reflects the current version.

## Reporting a Vulnerability
If you believe you’ve found a security issue:

1) Prefer **GitHub Security Advisories** (private disclosure) if available:
   Repo → **Security** → **Advisories** → **Report a vulnerability**
2) Otherwise contact me via LinkedIn:
   https://www.linkedin.com/in/badrkarim/

Please include:
- Description
- Reproduction steps (sanitized if needed)
- Impact
- Suggested mitigation

## Response
I aim to acknowledge reports quickly and follow up based on severity.
